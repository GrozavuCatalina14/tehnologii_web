
# Tehnologii Web – Proiect

# Descriere

Aplicația permite atribuirea de task-uri pentru fiecare angajat ;i adaugarea unor noi executanti de task-uri.

##Tehnologii folosite
- Frontend: React (SPA) + CSS
- Backend: Node.js + Express
- Bază de date: SQLite
- ORM: (Sequelize)

# Rulare locala

# Backend
cd backend
npm install
node server.js
